# Change Log

All notable changes to IPTV Mac Stalker for LG webOS will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-07-01

### Added
- Initial release of IPTV Mac Stalker for LG webOS
- Full Mac Stalker middleware protocol implementation
- Support for IPTV portal authentication
- Channel categorization and browsing
- TV remote control optimization
- Dark mode support
- Keyboard shortcuts and navigation
- CORS proxy support for web connections
- Editable playlist management
- Configuration import/export
- Connection status indicators
- Stream URL generation
- WebOS IPK packaging

### Features
- **Mac Stalker Protocol**: Complete implementation of handshake, token, and streaming
- **User Interface**: Modern React-based UI with TV remote optimization
- **Configuration**: Easy setup with MAC address and portal configuration
- **Streaming**: Support for live TV channels and VOD content
- **Categorization**: Browse channels by genre/category
- **Responsive**: Works on different screen sizes and orientations

### Supported Servers
- Stalker Middleware
- Infomir portals
- Xtream Codes compatible servers
- Other Mac Stalker protocol servers

### Documentation
- Comprehensive installation guide
- Developer documentation
- TV compatibility matrix
- Build scripts and packaging instructions
- License and legal information

### Technical Details
- Built with React 18 + TypeScript
- Styled with Tailwind CSS
- Packaged for LG webOS 3.0+
- Optimized for TV remote control
- Hardware accelerated streaming where available
